# World Cup 26 Predictor

A browser-based 2026 FIFA World Cup probability lab featuring:

- Match win, draw, and loss probabilities
- Poisson-based score predictions
- 5,000-run Monte Carlo tournament simulation
- Live fixtures and score updates
- Responsive desktop and mobile design

## Run locally

Serve the directory with any static web server, then open `index.html`.

## Data and limitations

The model combines strength seeds, World Cup history, experience, and host
advantage. Live fixtures and scores are sourced from ESPN's public scoreboard
endpoint. Results are exploratory estimates, not betting advice.
